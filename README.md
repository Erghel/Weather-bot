## Commands
- ``/start`` - bot launch    
- ``/help`` - bot commands    
- ``/donate`` - support developers    
- ``/weather`` - weather output in the specified city     
- ``/subscribe`` - weather subscription    
- ``/get_geo`` - get city coordinates    
- ``/uv_index`` - get the uv index    
- ``/credits`` - bot author    
- ``/report`` - report a bug
